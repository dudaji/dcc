### 과제

* 교재의 각 챕터 요약
* 본인이 면접관이라 생각하고 예상질문목록 만들어오기

* 스터디원들이 돌아가면서 각 챕터별로 핵심내용에 대하여 설명한다.
* 작성한 예상질문목록들을 가지고 서로 질문과 답변을 한다.
